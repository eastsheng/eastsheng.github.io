Linux Common Commands

Mr. C

Computer/Linux

发布于：2020年7月10日

0  

一、空间

  1. 查看空间大小：df -h；
  2. 查看该目录下文件大小：du -sh *

二、文件夹

  1. 创建文件夹：mkdir filename；
  2. 还可以同时创建两个文件夹：mkdir file1 file2；
  3. 创建一个包含路径的文件夹：mkdir -p /home/file/file1
  4. 删除文件；rm -f filename
  5. 删除空文件夹：rmdir foldername；
  6. 删除有文件的文件夹：rm -rf foldername；
  7. 同时删除两个有文件的文件夹：rm -rf folder1 folder2；
  8. 返回上一级目录：cd ..；
  9. 显示当前工作路径：pwd；
  10. 查看当前目录中的文件/文件夹ls；

三、解压

  1. 解压".tar.gz"：tar -zxvf filename.tar.gz;
  2. 解压".zip"：unzip filename.zip;

跑MD结构结构散了，常见原因总结：

1. 势函数的问题，调研一下该势函数是否有人模拟过成果过。
2. 原子结构不稳定导致的，例如：
  • 模拟 缺陷 的时候，缺陷过多导致结构不稳定；
  • 模拟 高温 或者 高压 环境下时，导致结构不稳定；
3. 势参数设置出错，例如：
  • data文件中原子类型与input文件中配置势参数时的原子类型顺序不一致；
3. 是否有足够的真空层
