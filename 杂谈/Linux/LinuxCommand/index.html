Linux Common Commands

一、空间

  1. 查看空间大小：df -h；
  2. 查看该目录下文件大小：du -sh *

二、文件夹

  1. 创建文件夹：mkdir filename；

  2. 还可以同时创建两个文件夹：mkdir file1 file2；

  3. 创建一个包含路径的文件夹：mkdir -p /home/file/file1

  4. 删除文件；rm -f filename

  5. 删除空文件夹：rmdir foldername；

  6. 删除有文件的文件夹：rm -rf foldername；

  7. 同时删除两个有文件的文件夹：rm -rf folder1 folder2；

  8. 返回上一级目录：cd ..；

  9. 显示当前工作路径：pwd；

  10. 查看当前目录中的文件/文件夹ls；

  11. 复制一个目录到另一个目录下:cp -r dir1 dir2

    三、解压

  12. 解压".tar.gz"：tar -zxvf filename.tar.gz;

  13. 解压".zip"：unzip filename.zip;

Alamode 运行

  • 执行alm
    alm alm.in > alm.log

    MODE = suggest的时候可以获得一个包含笛卡尔坐标的displacement patterns
    MODE = optimize的时候需要一个数据集（DFSET）,且得到xx.fcs(力常数),xx.xml(必要的信息)

  • 执行anphon
    anphon anphon.in > anphon.log
    or
    mpirun -np 16 anphon anphon.in > anphon.log

    MODE = phonons的时候perform (harmonic) phonon calculations
    MODE = RTA的时候conduct thermal conductivity calculations
